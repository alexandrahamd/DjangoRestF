Учебный DRF проект учебного портала.
Реализованы модели для: курсов, уроков, оплаты, подписки, логов оплаты
С помощью Celery отправляется сообщение, когда курс, на который он подписан, был изменен.
И так же с помощью Celery каждую секунду проверяется статус платежа.
запуск:
python manage.py runserver
celery -A mysite worker --loglevel=info
celery -A mysite beat -l info -S django
